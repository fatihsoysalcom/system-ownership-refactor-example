# System Ownership Refactor Example

This example demonstrates the principles of "ownership" in software development. It presents a simple inventory management system designed with clarity, robustness, and maintainability in mind, contrasting with a hypothetical "bomb-like" system. By encapsulating logic, validating inputs, and handling errors gracefully, it shows how a system can become reliable and easy to understand, moving from a fragile state to a "safe harbor."

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal using `python main.py`. Observe the output for successful and failed order processing, along with logging messages.
